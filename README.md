# 🎮 AI Game Strategist Ultimate

## 🚀 Overview

AI Game Strategist is an interactive simulation system that models intelligent decision-making in a game environment.
It evaluates user actions based on strategy and reasoning, assigns rewards, and provides AI-driven feedback.

---

## 🎯 Key Features

* 🎮 Interactive game simulation
* 🧠 Strategy-based decision evaluation
* 🏆 Reward system (dynamic scoring)
* 📈 Real-time score tracking
* 📊 Score visualization using graphs
* 🤖 AI-based move suggestions
* 🎯 Difficulty levels (Easy, Medium, Hard)

---

## 🧠 How It Works

1. User selects an action:

   * Attack
   * Defend
   * Collect resources

2. User provides explanation for the decision

3. System:

   * Updates game state
   * Calculates reward
   * Evaluates reasoning quality
   * Suggests next move

4. Displays:

   * Updated health & resources
   * Reward score
   * Total score progression
   * AI feedback & suggestions
